# Index in Angular Sankey component

A Sankey diagram visualizes the flow and quantity between nodes using nodes and links rendered in SVG. This index summarizes available topics and links to examples.

## Key Features

* **Flow Visualization** - Visualize directional flow with proportional link widths. Also: display weighted flows and relationships between categories.
* **Node Customization** - Control node size, color, and layout. Also: customize appearance, opacity, and positioning.
* **Link Styling** - Configure curvature, color blending and thickness. Additionally: control opacity levels and link color behavior.
* **Labels & Tooltips** - Add labels and contextual tooltips for nodes and links. Also: support interactive labels with customizable fonts, sizes, and padding.
* **Legend & Titles** - Add legends, titles and subtitles with styling. Additionally: interactive legends can highlight categories.
* **Orientation & RTL** - Support horizontal/vertical layouts and right‑to‑left rendering. Also: fully supports both flow orientations for Sankey diagrams.
* **Print & Export** - Export diagrams to PNG/JPEG/SVG/PDF formats.
* **Events & Rendering Hooks** - Use events and rendering callbacks for dynamic styling and interaction. Also: supports rich interactions including hover effects, click events, and item highlighting.
* **Themes** - Multiple built‑in themes for consistent visual styling.
