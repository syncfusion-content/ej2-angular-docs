# Release History

Each Syncfusion<sup style="font-size:70%">&reg;</sup> release introduces new features, enhancements, and updates. For a detailed overview of the changes included in each release, refer to the Syncfusion<sup style="font-size:70%">&reg;</sup> Angular [release notes](https://ej2.syncfusion.com/angular/documentation/release-notes/index/).

For comprehensive upgrade instructions and information, consult the [Upgrade Guide](https://help.syncfusion.com/upgrade-guide/angular-ui-components) for Angular. The guide provides details on the following:

- Breaking Changes

- Bug Fixes

- New Features

- Known issues between your current version and the latest version you are planning to upgrade to.
