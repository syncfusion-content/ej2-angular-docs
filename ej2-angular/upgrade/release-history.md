# Release History

Each Syncfusion<sup style="font-size:70%">&reg;</sup> release introduces new features, enhancements, and updates. For a detailed overview of the changes included in each release, refer to the Syncfusion<sup style="font-size:70%">&reg;</sup> Angular [release notes](https://ej2.syncfusion.com/angular/documentation/release-notes/index/).

For comprehensive upgrade instructions and information, consult the [Upgrade Guide](https://help.syncfusion.com/upgrade-guide/angular-ui-components) for Angular UI Components. The guide provides details on the following:

* Breaking changes between versions
* Bug fixes and improvements
* New features and enhancements
* Known issues between your current version and the latest version you are planning to upgrade to
## Accessing Release Information

To stay informed about Syncfusion® releases:

* Review the [Release Notes](https://ej2.syncfusion.com/angular/documentation/release-notes/index/) for detailed changelogs organized by version
* Check the [Product Release Lifecycle](https://www.syncfusion.com/support/product-lifecycle) page for support timelines and release schedules
* Subscribe to Syncfusion® announcements for timely updates on new releases
