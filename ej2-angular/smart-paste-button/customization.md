# Types and Appearance

The Syncfusion Angular Smart Paste Button component fully inherits all the properties, types, and styling options of the Syncfusion Angular Button component. This means that you can leverage the existing features of the Syncfusion Angular Button while benefiting from the enhanced functionality of the Smart Paste Button.
