# Globalization

Globalization combines two key capabilities: adapting controls to various languages through date and number formatting (Internationalization), and translating text content with cultural customizations (Localization). Syncfusion<sup style="font-size:70%">&reg;</sup> Angular UI components are configured for the `en-US` locale by default.

Globalization in Syncfusion<sup style="font-size:70%">&reg;</sup> Angular is achieved through:

* [Localization](https://ej2.syncfusion.com/angular/documentation/common/globalization/localization) - Customize and translate text content in Syncfusion<sup style="font-size:70%">&reg;</sup> Angular UI Components to meet specific cultural requirements.

* [Internationalization](https://ej2.syncfusion.com/angular/documentation/common/globalization/internationalization) - Format and parse date and number objects in Syncfusion<sup style="font-size:70%">&reg;</sup> Angular UI Components according to different cultural conventions.
