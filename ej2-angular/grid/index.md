# Index in Angular Grid component

The Grid component is a powerful control for displaying and manipulating tabular data with comprehensive configuration options to control how data is presented and manipulated. It seamlessly pulls data from various sources, including arrays of JavaScript objects, **OData web services**, or **DataManager**, and binds data fields to columns while displaying column headers to identify fields with full support for grouped records.

The Grid component provides essential data management capabilities including paging, sorting, filtering, searching, and grouping, making it ideal for building data-driven applications with rich user interactions.

## Key Features

### **Data Management**
* [**Data sources**](./data-binding) - Bind the Grid component with an array of JSON objects, remote services, or DataManager for flexible data handling.
* [**Sorting**](./sorting) and [**grouping**](./grouping) - Support n-levels of sorting and grouping with intuitive user interactions.
* [**Filtering**](./filtering) - Comprehensive filtering options including filter bar, menu, Excel-style, and checkbox filters at each column level.
* [**Searching**](./search) - Built-in search functionality to quickly locate specific data across all columns.
* [**Paging**](./paging) - Efficient pagination with customizable page sizes and navigation controls for handling large datasets.

### **Data Operations**
* [**Editing**](./edit) - Complete CRUD operations (Create, Read, Update, Delete) with multiple edit modes including inline, dialog, and batch editing.
* [**Selection**](./selection) - Flexible row and cell selection with support for single, multiple, and programmatic selection modes.
* [**Clipboard**](./clipboard) - Copy selected rows and cells from the grid for seamless data transfer.
* [**Aggregation**](./aggregates) - Built-in aggregation functions to summarize column data with custom aggregate support.

### **Column Features**
* [**Columns**](./columns) - Advanced column definitions that serve as the dataSource schema with extensive formatting and display options.
    * [**Reordering**](./columns/#reorder) - Drag and drop functionality to reposition columns dynamically within the grid header.
    * [**Column Chooser**](./columns/#column-chooser) - Interactive column visibility toggle with checkbox-based selection interface.
    * [**Resizing**](./columns/#column-resizing) - Dynamic column width adjustment by dragging column header borders.
    * [**Freeze**](./scrolling/#frozen-rows-and-columns) - Lock columns and rows in place for easy data comparison while scrolling.
    * [**Cell Spanning**](./columns/#column-spanning) - Merge grid cells across rows or columns based on custom criteria.
    * [**Foreign Key Column**](./columns/#foreign-key-column) - Display values from external data sources using foreign key relationships.
    * [**Cell Styling**](./how-to/#customize-column-styles) - Comprehensive styling options using CSS classes or programmatic approaches.
    * [**Templates**](./columns/#column-template) - Custom rendering with templates for headers, cells, and edit forms to create rich user experiences.

### **User Interface & Experience**
* [**Context Menu**](./context-menu) - Right-click context menus for cells, headers, and pager with customizable action lists.
* [**Toolbar**](./tool-bar) - Configurable toolbar with built-in and custom actions for enhanced user productivity.
* [**Responsive Design**](./adaptive) - Adaptive UI that automatically adjusts to different screen sizes and devices.
* [**Keyboard Navigation**](./accessibility/#keyboard-interaction) - Full keyboard accessibility with intuitive navigation and shortcuts.

### **Export & Integration**
* [**Export**](./pdf-export) - Export grid data to multiple formats including Excel, PDF, and CSV with customization options.
* [**Print**](./print) - Print functionality with customizable print layouts and formatting.

### **Globalization & Accessibility**
* [**RTL Support**](./global-local/#right-to-left---rtl) - Complete right-to-left text direction support for international applications.
* [**Localization**](./global-local/#localization) - Built-in internationalization support for UI text and date/number formatting.
* [**Accessibility**](./accessibility) - WCAG 2.0 compliant with screen reader support and keyboard navigation.

### **Performance & Optimization**
* [**Virtual Scrolling**](./virtual) - Handle large datasets efficiently with on-demand loading and rendering.
* [**Infinite Scrolling**](./infinite-scroll) - Seamless data loading as users scroll through large datasets.
* [**Column Virtualization**](./columns/#column-virtualization) - Optimize performance when working with numerous columns.
