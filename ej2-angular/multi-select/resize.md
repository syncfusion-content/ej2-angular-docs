# Resizing in Angular MultiSelect component

You can dynamically adjust the size of the popup in the MultiSelect component by using the [AllowResize](https://ej2.syncfusion.com/angular/documentation/api/multi-select/#allowresize) property. When enabled, users can resize the popup, improving visibility and control, with the resized dimensions being retained across sessions for a consistent user experience.
